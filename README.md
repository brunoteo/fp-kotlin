# Applied Functional Programming Workshop - Kotlin Edition

## Gradle commands

Run build

```sh
$ ./gradlew build
```

Run cli application

```sh
$ ./gradlew app:run
```

Run web application

```sh
$ ./gradlew webapp:run
```
